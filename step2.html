INSERTING OUR GAME CHARACTER

To get started, we are going to create an object representing the main character and name it player. As you look through the code that follows, keep in mind that there are lots of different ways to accomplish just about everything in JavaScript and this is just one way to get things done. In fact, this is not the best way for most games, but is a very simple and short method that keeps the file size of the game low.

  1. Open game.js in your text editor
  2. Locate the following line of code: var GameLoop =  function(){ (it is near the bottom)...
  3. Click on the blank line ABOVE the code indicated above to place the insertion point (the cursor) there...
  4. Copy and paste the following code onto the blank line:
    var player = new (function(){
    //Create a new object based on a function and assign
    //what it returns to the 'player' variable
    var that = this;
    //'that' will be the context now
       that.image = new Image();
       that.image.src = "angel.png"
    //Create new Image and set it's source to the 'angel.png' image
       that.width = 65;
    //Width of a single frame
       that.height = 95;
    //Height of a single frame
     
       that.X = 0;
       that.Y = 0;
    //X & Y position

Notice that the above code defines the angel.png image as the image we want to use as our character in the game. It also defines the size of the frame that the image will appear in and sets its X and Y position within the game.
